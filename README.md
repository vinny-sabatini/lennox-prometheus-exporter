# Lennox-Prometheus-Exporter

Prometheus exporter for Lennox thermostat metrics.

This is a [Prometheus exporter](https://prometheus.io/docs/instrumenting/exporters/) intended to expose Lennox iComfort API on your local network.

The initial implementation is based on the conversation in [PeteRager/lennoxs30api](https://github.com/PeteRager/lennoxs30api/issues/62).

This has been tested using a single zoned S40.
